# gridPython

PythonMPI support with a grid environment so that jobs can be launched much easier
<br>
There are a couple of requireemnts to use the gridPython package
<br>
<li>
It assumes that PythonMPI is distributed with gridPython at its root directory. <br>
</li>
<li>
It is also assumes that a local configuration is available from $HOME/pythonmpi directory.
</li>
<br>
<br>
An example run script is available in the example directory. 
The GRIDPYTHON_HOME environment in the example run script needs to be modified according to the user runtime environment.
<br>
