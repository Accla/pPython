Usage
-----

* Running an example on LLGrid
-------------------------------

1. Go to an example directory

   cd $PPYTHON_HOME/examples/Param_Sweep

2. Edit the run_on_llsc.py script for your LLGrid account.

   Modify the following variables:
   PPYTHON_HOME = "/path/to/your/gridPython/installation/directory
   HOME_PATH = "/home/gridsan/LLID" (replace LLID with your actual LLID)

   Following variables can be modified to change the number of Python MPI processes:
   n_proc

3. Load the corresponding Anaconda module as defined during the installation 
   via the pyMPI_Comm_settings_local.py file.

   module load anaconda/2021b

4. Launch the job

   python run_on_llsc.py
