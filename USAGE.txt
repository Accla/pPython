Usage
-----

* Running an example on LLGrid
-------------------------------

1. Go to an example directory

   cd $PPYTHON_HOME/examples/Param_Sweep

2. Load the corresponding Anaconda module as defined during the installation 
   via the pyMPI_Comm_settings_local.py file.

   module load anaconda/Python-ML-2024b 

3. Launch the job

   python RUN.py (use the pPYthon package installation with pip)yypl
   python RUN_on_src.py (use the pPYthon source installation)
